###作品説明
背景にマスコットキャラクターの画像が表示されるアナログ時計です。  
ウィンドウのどこかをクリックすると背景画像を切り替えることが出来ます。

"Acfcw.zip"ファイルを解凍し、お使いのwindowsのビット数のフォルダ内にある"watch.exe"を実行して下さい。
